## ğŸš€ Getting Started

### Prerequisites

- Python (see `.python-version` for specific version)
- pip or uv package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/WillowsCosmic/Serenity.git
cd Serenity
```

2. Install dependencies:
```bash
pip install -r pyproject.toml
# or if using uv
uv sync
```

3. Set up environment variables for:
   - Google Gemini AI API key
   - Twilio credentials for emergency services

4. Run the application:
```bash
streamlit run frontend.py
# or
python main.py
```
